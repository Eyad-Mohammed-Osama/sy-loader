sy-loader

This Library Uses A Set Of HTML5 data-* Attributes,And CSS3 Animation Properties To Control The Loader.

  • The data-sy-loader Attribute

    • This Attribute Is Required,And Should Contain The Value "true",Otherwise The Loader May Not Appear.
    In Next Versions,I Will Try To Remove This Property And Use A Custom Tag Instead.

  • The data-sy-loader-border-type Attribute

    • This Attribute Is Required,And Should Contain One Of CSS Border Style Names: Solid,Dotted,Dashed,Inset,Outset,Ridge,Hidden,None,Double,Groove.
    Note:

    This Attribute Doesn't Have A Default Value.
    Omitting,Or Setting A Wrong Value To This Attribute May Lead To An Alarm On Your Page Each Time You Load It.

  • The data-sy-loader-border-width Attribute

    • This Attribute Is Optional ,And If You Omitted It,The Loader's Border Will Have A Default Width Of 10px.
    You May Specify Any Width You Like,And All CSS Units Are Supported.

  • The data-sy-loader-border-color Attribute

    • This Is A Required Attribute.
    Defines The Border Color For The Loader.
    You May Use The Hexadecimal Colors,Or The RGB() Colors,Or Specify Colors By Their Names.

  • The data-sy-loader-spinner-color Attribute

    • This Attribute Is Required.
    Defines Color For That Spinning Thing Inside The Loader.
    You May Use The Hexadecimal Colors,Or The RGB() Colors,Or Specify Colors By Their Names.

  • The data-sy-loader-radius Attribute

    • This Attribute Is Optional.
    Specify The Radius For Loader (Border Width Isn't Included),Since The Loader Is A Circle.

  • The data-sy-loader-duration Attribute

    • This Attribute Is Optional.
    Specifies The Required Time For The Loader To Complete A 360Â° Degree Turn.
    If You Omitted This Attribute,The Default Required Time Is Two Seconds.
